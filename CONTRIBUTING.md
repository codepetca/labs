# Contributing

Before submitting work to Codepet Labs, read [`docs/onboarding.md`](docs/onboarding.md)
and [`docs/builder-expectations.md`](docs/builder-expectations.md).

Do not open a pull request with code, docs, designs, media, generated assets, or
demo material unless you are allowed to share that work with Codepet Labs.

Keep contributions small, use mock data, and explain what changed, how it was
checked, and what remains unfinished.
