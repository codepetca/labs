# Labs Page Structure

Use this as the common structure for each student's personal Labs page.

The page should feel like a CodePet Labs builder log, not a generic homepage.
Keep it short, current, and easy to update every week.

## Shared Header

```text
CodePet Labs - Summer 2026
AI-assisted builder log
```

## Page Sections

```md
# Name

## Current Focus
A short sentence about what you are building or exploring right now.

## This Week
- What changed
- What you tried
- What you published or tested

## AI Help
- Tool used
- What it helped with

## Next Week
- The next small thing you plan to build, improve, or learn

## Links
- Vercel:
- GitHub:
- Pull request or demo:
```

## Footer

```text
Built as part of CodePet Labs Summer 2026.
```

Students can change the layout and style, but the page should keep the shared
Labs identity and weekly-update rhythm.
