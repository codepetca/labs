# Vision

Codepet Labs is an independent AI-native student software lab connected to the Pika ecosystem.

The lab exists to help motivated students learn modern software development by building useful, visible, Pika-adjacent tools. It is not a school club, a formal class, an employment program, or a path to unsupervised production access.

The core idea is simple:

- Students learn by shipping small projects.
- AI is used as a serious development tool, not a shortcut around understanding.
- Pika gives the work a real ecosystem to orbit.
- Pika core stays supervised and protected.
- Student work should create attribution and portfolio value, even when the code is later rewritten or replaced.

Labs should feel practical, grounded, and builder-focused. The standard is not perfection. The standard is visible progress, clear thinking, and responsible engineering habits.
